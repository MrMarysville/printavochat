# contactCreate

Create a contact

## Input fields

-   **id** (`ID!`)
    ID of the Customer to whom this Contact belongs

-   **input** (`ContactInput!`)
    Create contact values

## Return fields

-   **customer** (`Customer!`)
    This contact's customer

-   **email** (`String`)
    Email

-   **fax** (`String`)
    Fax

-   **firstName** (`String`)
    First name

-   **fullName** (`String`)
    Full name

-   **id** (`ID!`)
    The ID

-   **lastName** (`String`)
    Last name

-   **orderCount** (`Int`)
    Order count

-   **phone** (`String`)
    Phone

-   **timestamps** (`ObjectTimestamps!`)
    Object timestamps
