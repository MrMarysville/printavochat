# deliveryMethodArchive

Archive/restore a delivery method

## Input fields

-   **archived** (`Boolean!`)
    Is the delivery method archived?

-   **id** (`ID!`)
    ID of delivery method to archive/restore

## Return fields

-   **archived** (`Boolean!`)
    Archived?

-   **id** (`ID!`)
    The ID

-   **name** (`String!`)
    Delivery type

-   **timestamps** (`ObjectTimestamps!`)
    Object timestamps
