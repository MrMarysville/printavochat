# contact

Get contact by ID

## Arguments

-   **id** (`ID!`)
    The ID of the contact to find

## Return fields

-   **customer** (`Customer!`)
    This contact's customer

-   **email** (`String`)
    Email

-   **fax** (`String`)
    Fax

-   **firstName** (`String`)
    First name

-   **fullName** (`String`)
    Full name

-   **id** (`ID!`)
    The ID

-   **lastName** (`String`)
    Last name

-   **orderCount** (`Int`)
    Order count

-   **phone** (`String`)
    Phone

-   **timestamps** (`ObjectTimestamps!`)
    Object timestamps
