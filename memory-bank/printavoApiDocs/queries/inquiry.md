# inquiry

Get an Inquiry by ID

## Arguments

-   **id** (`ID!`)
    Inquiry ID to get

## Return fields

-   **email** (`String`)
    Email of person making the inquiry

-   **id** (`ID!`)
    The ID

-   **name** (`String!`)
    Name of person making the inquiry

-   **phone** (`String`)
    Phone number of person making the inquiry

-   **request** (`String`)
    Request

-   **timestamps** (`ObjectTimestamps!`)
    Object timestamps

-   **unread** (`Boolean!`)
    Has the inquiry been read?
